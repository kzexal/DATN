# DANH MỤC CÁC CHỮ VIẾT TẮT

Dưới đây là bảng danh mục các chữ viết tắt được sử dụng trong đồ án tốt nghiệp **"Hệ thống điểm danh và giám sát hành trình xe buýt học sinh thông minh sử dụng công nghệ IoT"**, được sắp xếp theo thứ tự chữ cái (A-Z) để đảm bảo tính chuyên nghiệp và dễ tra cứu:

| Chữ viết tắt | Từ tiếng Anh đầy đủ | Nghĩa tiếng Việt / Diễn giải |
| :--- | :--- | :--- |
| **ADC** | Analog-to-Digital Converter | Bộ chuyển đổi tương tự sang số |
| **API** | Application Programming Interface | Giao diện lập trình ứng dụng |
| **AT (Commands)**| Attention Commands | Tập lệnh điều khiển modem (Tập lệnh AT) |
| **BDS** | BeiDou Navigation Satellite System | Hệ thống định vị vệ tinh Bắc Đẩu (Trung Quốc) |
| **BOM** | Bill of Materials | Danh mục linh kiện / Vật tư thiết kế |
| **CSQ** | Signal Quality | Chỉ số đo chất lượng tín hiệu sóng di động |
| **DAC** | Digital-to-Analog Converter | Bộ chuyển đổi số sang tương tự |
| **EMI** | Electromagnetic Interference | Nhiễu điện từ trường |
| **ESP32** | - | Họ vi điều khiển/SoC tích hợp Wi-Fi & Bluetooth của Espressif |
| **FreeRTOS** | Free Real-Time Operating System | Hệ điều hành thời gian thực miễn phí cho vi điều khiển |
| **GND** | Ground | Cực âm / Nối đất chung |
| **GPIO** | General Purpose Input/Output | Cổng vào/ra dữ liệu đa dụng của vi điều khiển |
| **GPS** | Global Positioning System | Hệ thống định vị toàn cầu (Mỹ) |
| **HTTP** | Hypertext Transfer Protocol | Giao thức truyền tải siêu văn bản |
| **HTTPS** | Hypertext Transfer Protocol Secure | Giao thức truyền tải siêu văn bản bảo mật |
| **I2C / I²C** | Inter-Integrated Circuit | Chuẩn giao tiếp song song giữa các vi mạch |
| **IC** | Integrated Circuit | Mạch tích hợp (vi mạch) |
| **IoT** | Internet of Things | Vạn vật kết nối Internet |
| **JSON** | JavaScript Object Notation | Định dạng trao đổi dữ liệu văn bản gọn nhẹ |
| **LTE** | Long Term Evolution | Công nghệ truyền thông di động thế hệ thứ tư (4G LTE) |
| **MCU** | Microcontroller Unit | Khối điều khiển trung tâm / Vi điều khiển |
| **MISO** | Master Input Slave Output | Đường truyền dữ liệu từ tớ sang chủ (chuẩn SPI) |
| **MOSI** | Master Output Slave Input | Đường truyền dữ liệu từ chủ sang tớ (chuẩn SPI) |
| **NMEA** | National Marine Electronics Association | Chuẩn định dạng dữ liệu định vị vệ tinh |
| **OLED** | Organic Light Emitting Diode | Điốt phát quang hữu cơ (màn hình hiển thị) |
| **PCB** | Printed Circuit Board | Bảng mạch in |
| **PDP** | Packet Data Protocol | Giao thức dữ liệu gói trong mạng viễn thông |
| **RAM** | Random Access Memory | Bộ nhớ truy cập ngẫu nhiên |
| **RFID** | Radio Frequency Identification | Nhận dạng bằng tần số sóng vô tuyến |
| **ROM** | Read-Only Memory | Bộ nhớ chỉ đọc |
| **RST** | Reset | Thiết lập lại / Tái khởi động thiết bị |
| **RX / RXD** | Receive Data | Chân/Đường nhận dữ liệu (giao tiếp UART) |
| **SCL** | Serial Clock | Đường cấp xung nhịp nối tiếp (giao tiếp I2C) |
| **SCK / SCLK**| Serial Clock | Đường cấp xung nhịp nối tiếp (giao tiếp SPI) |
| **SDA** | Serial Data | Đường truyền dữ liệu nối tiếp (giao tiếp I2C) |
| **SIM** | Subscriber Identity Module | Thẻ nhận dạng thuê bao di động (thẻ SIM) |
| **SMD** | Surface-Mount Device | Linh kiện gắn trên bề mặt (linh kiện dán) |
| **SMS** | Short Message Service | Dịch vụ tin nhắn ngắn |
| **SPI** | Serial Peripheral Interface | Chuẩn giao tiếp ngoại vi nối tiếp |
| **SS / CS** | Slave Select / Chip Select | Chân chọn thiết bị tớ / Chọn chip (giao tiếp SPI) |
| **SSL** | Secure Sockets Layer | Lớp bảo mật cổng socket (mã hóa dữ liệu truyền) |
| **THT** | Through-Hole Technology | Công nghệ linh kiện xuyên lỗ (linh kiện cắm chân) |
| **TLS** | Transport Layer Security | Giao thức bảo mật tầng truyền tải |
| **TX / TXD** | Transmit Data | Chân/Đường truyền dữ liệu (giao tiếp UART) |
| **UC** | Use Case | Ca sử dụng (trong UML) |
| **UID** | Unique Identifier | Mã định danh duy nhất của thẻ RFID |
| **UML** | Unified Modeling Language | Ngôn ngữ mô hình hóa thống nhất |
| **UART** | Universal Asynchronous Receiver-Transmitter| Bộ truyền nhận dữ liệu dị bộ đa năng |
| **USB** | Universal Serial Bus - Cổng kết nối nối tiếp vạn năng | Cổng truyền thông nối tiếp vạn năng |
| **VCC** | Voltage Common Collector | Cực dương nguồn điện cấp cho linh kiện |
| **UTC** | Coordinated Universal Time | Giờ phối hợp quốc tế |
